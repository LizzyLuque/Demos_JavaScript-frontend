Demos JavaScript es un proyecto personal creado con la finalidad de mostrar las habilidades aprendidas en mis cursos de Udemy:
  • Máster en JavaScript: Aprender Js, jQuery, Angular, NodeJS
  • Máster en frameworks para JavaScript
  • Aplicaciones en tiempo real con sockets y rest.
  • JavaScript ES6 Course: ECMA Script 6 (Step by Step).
  • Angular 10 Course: Build Angular Apps (Step by Step).
  • Learn Bootstrap - For Beginners.
Demos JavaScript consta de 5 módulos, los cuales se describen a continuación:
  • Chat
  • Sistema de Colas
  • CRUD
  • Gráficas
  • Mapas

El módulo del chat contiene las siguientes funcionalidades:
  1. Sesión local y recuperación la misma utilizando LocalStorage
  2. Salas de chat
  3. Envío de mensajes privados
  4. Utilidades para el usuario (Emoticones y cambio de color en el nick)
Entre los lenguajes, frameworks y librerías utilizados para programar este módulo, destacan los siguientes:
  • FrontEnd: JavaScript, Typescript, Angular, HTML, CSS, Bootstrap, socket.io
  • BackEnd: NodeJS, Socket.io, express

El sistema de colas consta de las siguientes funcionalidades:
  1. Pantalla de atención al público
  2. Generación de tickets por área de atención
  3. Atención de tickets por cada escritorio
Entre los lenguajes, frameworks y librerías utilizados para programar este módulo, destacan los siguientes:
  • FrontEnd: JavaScript, Typescript, Angular, HTML, CSS, Bootstrap, socket.io
  • BackEnd: NodeJS, Socket.io, express

El módulo llamado CRUD contiene las siguientes funcionalidades:
  1. Autenticación (registro de usuarios e inicio de sesión)
  2. Resguardo de paginas sensibles mediante sesión local y recuperación la misma utilizando LocalStorage
  3. Un crud de artículos
  4. Consumo de APIs con autorización via token jwt
Entre los lenguajes, frameworks y librerías utilizados para programar este módulo, destacan los siguientes:
  • FrontEnd: JavaScript, Typescript, Angular, HTML, CSS, Bootstrap, moment
  • BackEnd: NodeJS, express, mongoose, JWT

Las funcionalidades del módulo de gráficas son:
  1. Gráfica de linea que muestra las ventas en tiempo real
  2. Gráfica de barras que muestra los resultados de una encuesta en tiempo real
  3. Pantallas para alimentar los datos de las gráficas
Entre lenguajes, frameworks y librerías utilizados para programar este módulo, destacan los siguientes:
  • FrontEnd: JavaScript, Typescript, Angular, HTML, CSS, Bootstrap, socket.io, ng2-charts
  • BackEnd: NodeJS, Socket.io, express

EL módulo Mapas consta de:
  1. Generación de marcadores
  2. Movimiento de los marcadores
  3. Las anteriores funcionalidades se notifican en tiempo real al restos de los clientes conectados, mismo ejemplo utilizand MapBox y Google Maps
